# Auto Craft AR

A modern web application that provides access to AR (Augmented Reality) car models through QR code scanning and direct web links.

## Features

- **QR Code Integration**: Scan the QR code to access AR models on mobile devices
- **Direct Web Access**: Click the button to open AR models in web browsers
- **Responsive Design**: Works seamlessly on desktop and mobile devices
- **Modern UI**: Clean, professional interface with smooth animations

## Technology Stack

- HTML5
- CSS3 (with modern features like Flexbox and CSS Grid)
- Font Awesome Icons
- External AR platform integration

## Deployment

This project is configured for easy deployment on Netlify:

1. Connect your GitHub repository to Netlify
2. Set the build command to: `echo "No build required"`
3. Set the publish directory to: `.`
4. Deploy!

## Usage

1. **Desktop Users**: Click the "Click Here" button to open the AR model in a new tab
2. **Mobile Users**: Scan the QR code with your device's camera to access the AR experience
